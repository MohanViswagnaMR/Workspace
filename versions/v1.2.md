# Workspace — Version 1.2

**Release date:** 2026-05-29
**Branch:** main

---

## Overview

Workspace is a Notion-style connected workspace built with **Vite 6 + React 18 + Firebase 11**. It runs entirely in the browser — no backend required in local mode — and optionally syncs to Firestore for multi-device cloud access. Everything in the UI is a composable block: text, headings, lists, toggles, databases, images, code, file attachments, and more.

## Storage Modes

|                  | **Local Mode** (default)                    | **Cloud Mode**                              |
|------------------|---------------------------------------------|---------------------------------------------|
| **Trigger**      | `.env` empty or missing                     | Valid `VITE_FIREBASE_*` keys in `.env`      |
| **Login**        | None — opens straight to workspace          | Email/password or Google sign-in            |
| **Data store**   | Browser `localStorage`                      | Firestore (`workspaces/{uid}`)              |
| **Devices**      | This browser only                           | Synced across every device you sign in on   |
| **File system**  | File System Access API (optional)           | File System Access API (optional)           |

Writes are debounced by 700 ms so rapid edits coalesce into one Firestore write.

## Feature Set

### Block Editor
- **Text** — plain paragraph with inline formatting (bold, italic, underline, strikethrough, inline code)
- **Headings** — H1, H2, H3 with auto-sizing
- **Lists** — bulleted, numbered, to-do (checkbox), toggle (collapsible)
- **Quote** — blockquote with accent border
- **Callout** — highlighted box with emoji icon and selectable background color
- **Divider** — horizontal rule
- **Code block** — syntax-labelled editor with traffic-light header, custom language dropdown, and one-click copy
- **Image block** — upload from disk, paste URL, or pick from storage; inline caption editing
- **File attachment** — attach any file type; color-coded icon chip (red = PDF, purple = image, amber = video, blue = text); preview modal; download action
- **Web bookmark** — save a URL as a card with title and description
- **Sub-page** — embed a child page inline
- **Inline database** — embed any database view directly inside a page

### Slash Command Menu
Type `/` on any line to open the block picker. 20+ block types across Basic, Database, and Media groups. Keyboard-navigable with arrow keys, filtered by typing.

### Page Features
- Emoji icon picker (random, search, categorised)
- Gradient cover image picker (12 presets)
- Page title with auto-resize
- Drag-and-drop block reordering
- Block context menu (turn into, color, duplicate, delete, copy link)
- Undo / redo (structural history)
- Inline text color and background color per block
- Tab / Shift+Tab to indent/outdent blocks

### Databases (Multi-view)
- **Table view** — spreadsheet-style with add row/column
- **Board (Kanban) view** — drag cards between status columns
- **Gallery view** — image-led card grid
- **List view** — compact row list
- **Calendar view** — events on a monthly grid
- **Properties** — title, text, status, select (with colors), date, person
- Filter, sort, and group by any property
- Row peek modal for full record editing

### Navigation & Search
- Sidebar with nested page tree (infinite depth)
- Drag-and-drop page reordering and nesting
- Favorites section for pinned pages
- Trash with restore and permanent delete
- Instant full-text search (`Ctrl/⌘ + K`)
- Breadcrumb navigation in the topbar
- Back / forward history navigation

### Templates
- 12 built-in templates (Meeting Notes, Project Plan, Reading List, Weekly Review, etc.)
- One-click apply; templates insert a full block structure

### Sharing & Collaboration (Cloud Mode)
- Invite users by email with view or edit permission
- Shared panel shows active collaborators per document
- Real-time notification badge for incoming shares
- Workspace switcher — personal and shared workspaces

### Storage Page
- Dedicated storage manager for all uploaded files
- Three view modes: grid, large gallery, list
- Color-coded file type icons (matches attachment chip colors)
- Preview modal with prev/next navigation
- Download and delete per file
- Storage usage meter in the topbar

### Settings
- Light / dark mode toggle (`Ctrl/⌘ + Shift + L`)
- 7 accent color themes: Indigo, Blue, Ocean, Forest, Rose, Sunset, Violet
- Account management (cloud mode)
- Workspace management (rename, member list)
- Keyboard shortcuts reference
- Tutorial tour (restartable)

## Design System

### Tokens
All colors, spacing, and shadows are CSS custom properties on `:root` with a `.dark` override class. Accent themes layer on top via `.t-{name}` classes.

### Accent Themes
`indigo` · `blue` · `ocean` · `forest` · `rose` · `sunset` · `violet`

### Component Classes (key)
| Class            | Purpose                                      |
|------------------|----------------------------------------------|
| `.blk`           | Block wrapper (drag, gutter, body)           |
| `.b-{type}`      | Block type variant (b-code, b-image, etc.)   |
| `.fc-chip`       | File attachment chip                         |
| `.b-code`        | Code block container                         |
| `.code-head`     | Code block header bar                        |
| `.code-lang-sel` | Custom language dropdown                     |
| `.db`            | Database block wrapper                       |
| `.pop`           | Popup / floating menu                        |
| `.modal`         | Centered overlay modal                       |
| `.img-picker`    | Image picker (upload / embed / storage)      |
| `.file-picker`   | File picker (upload / from storage)          |
| `.storage-page`  | Full-page storage manager                    |
| `.sidebar`       | Left navigation panel                        |
| `.topbar`        | Top navigation bar                           |

## What's New in v1.2

### File Attachment
- Added bottom padding to the file attachment block so it no longer sits flush against adjacent text blocks
- Icon inside the chip is now a **rounded square** with proper margin instead of a flush left block
- Removed the dividing border between icon and filename; spacing handles the separation naturally

### Storage Page Icons
- All three storage views (grid, large gallery, list) now use the same **color-coded icon backgrounds** as the attachment chip:
  - 🔴 Red — PDF
  - 🟣 Purple — Images (JPG, PNG, etc.)
  - 🟡 Amber — Video
  - 🔵 Blue — Text files
  - ⚫ Gray — Other / unknown
- Icons have matching rounded corners in each view size

### Image Picker
- Removed the tab bar (Upload / From Storage / Embed link)
- **Upload** and **Embed link** are now clean bordered buttons pinned to the **bottom** of the picker
- Storage grid or drop zone fills the main body area
- Added margin inside the image block so the picker has breathing room from the block edges

### Code Block — Full Redesign
- **Traffic-light dots** (red / amber / green) in the header for a terminal-style look
- Native `<select>` replaced with a custom **CodeLangSelect** dropdown — styled pill trigger, animated popover with checkmark on the active language, matches the app's menu design language
- **Copy** is now a proper bordered button with an icon
- Code area uses `white-space: pre` for correct indentation rendering
- Header uses `align-items: center` so all elements sit on the same baseline
- Slightly differentiated background between header and code body
